# AI-NaMo: Secure Identity Verification

This is a Next.js application for AI-NaMo, a free AI-powered identity verification platform for verifying government-issued ID documents.

## Core Features

- **Secure User Authentication**: User registration and login with robust security measures.
- **Two-Factor Authentication**: Email-based 2FA for enhanced account security.
- **AI-Powered Document Verification**: Upload government-issued IDs and get AI-driven authenticity analysis, including confidence scores and real/fake classification.
- **Dashboard & History**: A comprehensive user dashboard to manage uploads, view verification results, and track your verification history.
- **Usage Statistics**: Visualize your verification activity with detailed charts and graphs.
- **AI Chatbot**: Get instant assistance and answers to your questions through an integrated AI chatbot.

## Getting Started

To get started, run the development server:

```bash
npm run dev
```

Then, open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
